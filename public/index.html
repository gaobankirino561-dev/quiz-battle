オンライン対戦ルール

基本ルール

  • 相手のHPを0にすると勝利です。
  • 一番最初に正解した人が、相手にダメージを与えます。
  • 自分のHPが0になると脱落（負け）です。

2人対戦の仕組み

  • 正解すると相手にダメージを与えます。

3人対戦の仕組み

  • 正解者は自分以外の全員にダメージを与えます。
  • 自分が脱落しても、勝負がつくまで観戦できます（リプレイ視聴）。

ダメージ計算

  • 基礎ダメージ：EASY＝2, NORMAL＝4, HARD＝7
  • スピードボーナス：
    • 0〜2秒以内：+3ダメージ
    • 2〜3秒以内：+1ダメージ
    • 3秒以降：ボーナスなし
  • 例：HARDで1.5秒で正解 → 7 + 3 = 10ダメージ

ルーム名: Room

  • 両者正解: 回答時間が速いプレイヤーが、遅いプレイヤーにダメージを与えます。
  • 片方正解: 正解者が不正解者にダメージを与えます。
  • 両者不正解: どちらもダメージを受けません。

3人対戦の仕組み（バトルロイヤル）

最も速く正解した人（1位）が攻撃側となり、他のプレイヤーがダメージを受けます。

  • 正解者が3人: 2位は50%、3位は100%のダメージを受けます。
  • 正解者が2人: 2位は50%、不正解者は150%のダメージを受けます。
  • 正解者が1人: 不正解者（2名）は150%のダメージを受けます。
  • 全員不正解: 全員ノーダメージです。

ダメージ計算

ダメージ ＝ 基礎ダメージ ＋ スピードボーナス

基礎ダメージ (難易度別)
  • EASY: 2点
  • NORMAL: 4点
  • HARD: 7点
スピードボーナス
  • 0〜2秒で回答: +3点
  • 2〜3秒で回答: +1点
  • 3秒以上: +0点

      ソロモード：一人で遊ぶ練習用モードです

      ソロモードでは、一人で練習できるタイムアタックなどのモードを遊ぶことができます。

      タイムアタック

      制限時間内にできるだけ多くの問題を解いてスコアを競うモードです。

      タイムアタックの詳しいルール

      • 制限時間内に、出題される問題にできるだけ多く正解すると高スコアになります。
      • 正解するとスコアが加算され、不正解やスキップはスコア加算なし（ペナルティは後で調整可能）。
      • 時間切れになるとその時点でゲーム終了し、最終スコアが表示されます。

      ※ ルール内容は今後の実装で調整・拡張される可能性があります。

      スピードラン

      目標正解数に到達するまでのタイムを競うモードです。

      スピードランの詳しいルール

      • 設定した目標正解数に達するまで問題を解き続け、経過時間を計測します。
      • タイムはストップウォッチ方式でカウント（問題回答中のみ加算）。
      • スコアは 100 + (目標正解数 × 2 - クリアタイム) で算出します。

      ※ ルール内容は今後の実装で調整・拡張される可能性があります。

      エンドレス

      ライフが尽きるまで問題を解き続けてスコアを稼ぐモードです。

      エンドレスの詳しいルール

      • ライフが0になる、または選択したジャンルの問題を出題しきったら終了です。
      • 正解でスコア加算（難易度によって加算量が変化）。
      • 5連続正解ごとにコンボボーナスが加算されます。

      ※ ルール内容は今後の実装で調整・拡張される可能性があります。

      タイムアタックのルール設定

      制限時間

      秒

      タイムアタック全体で使える時間です。問題に回答している間だけ減少します。

      ジャンル

      オンライン対戦と同じジャンルコードを使用してください（必要に応じて value は既存実装に合わせて調整する）。

      難易度

      オンライン対戦で使っている難易度コード（例: EASY / NORMAL / HARD）に揃える。

      スピードランのルール設定

      目標正解数

      この正解数に到達するまでの時間を計測します。

      ジャンル

      オンライン対戦と同じジャンルコードを使用してください（必要に応じて value は既存実装に合わせて調整する）。

      難易度

      オンライン対戦で使っている難易度コード（例: EASY / NORMAL / HARD）に揃える。

      エンドレスのルール設定

      ライフ
